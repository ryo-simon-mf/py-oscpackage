# py-oscpackage
[![](https://img.shields.io/badge/python-3+-blue.svg)](https://www.python.org/download/releases/3/)

## Requirement
- sys
- argparse
- pythonosc

## Install
```bash
$ git clone https://github.com/ryo-simon-mf/py-oscpackage.git
$ cd py-oscpackage
$ pip3 install -r requirements.txt
```
